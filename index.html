comando de git

  • git checkout -- index.html
  • git commit --amend
  • git reset HEAD index.html, elimina la modificacion del stage
  • git reset fei3429, borra el commit y deja en el local
  • git reset --hard fe234u3, borra el commit y deja todo limpio
  • git reset --soft fe234u3, borra el commit y deja en el stage
